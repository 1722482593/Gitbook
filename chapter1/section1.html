1.1 环境1工具安装

在Windows环境下的”可见即所得“工作方式所需工具安装

1、 在windows 下安装 nodejs

官网：https://nodejs.org/en/

2、Gitbook 安装

可以在命令行中输入下面的命令进行安装：

npm install gitbook-cli -g

3、Gitbook Editor 安装

下载Gitbook Editor安装，安装完打开客户端后，提示登录Gitbook账号，不需要登录。

4、安装 Git

可通过地址https://git-scm.com/downloads下载，在命令行下运行

git config --golbal user.email "you@example.com"

4、安装 gh-pages 工具

npm install -g gh-pages
