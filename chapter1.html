1、工具安装

工具安装

根据环境和个人使用习惯的不同，所需要安装的工具会有所不同。在这里我们准备在Windows环境下，建立一种比较简单的方式，书籍编写者可以在”可见即所得“方式下编写书籍，并可以方便地进行共享和协同写作。

为此，我们需要做一些准备。需要安装：

  • GitBook
  • Gitbook Editor
  • Git
  • gh-pages

至于在其他环境下以及用其他方式编辑书籍的情况，所需要安装的工具会在今后用专门章节补充。
